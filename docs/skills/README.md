# Plugin Skills Catalog

This directory contains the plugin-development skill catalog used for concrete plugin work in `agent-plugins`.

The main `agent` repository remains the source of truth for:

- plugin contract;
- governance constraints;
- platform-side orchestration and registry behavior;
- the boundary between catalog ideas, plugin packages, runtime modules, and skills.

Use this catalog for day-to-day plugin authoring in this repository.

## Available Skills

- `plugin-catalog-preflight`
- `plugin-scaffold`
- `plugin-manifest-edit`
- `plugin-lifecycle-implement`
- `plugin-test`
- `plugin-validate`
- `plugin-package`
- `plugin-deploy-prepare`
- `plugin-docs-update`
